# Development Changelog

## v0.0.0.1 (Alpha)

- Initial alpha release
- PC (PansiluBot) and Mobile (MethmiBot) crawler modes
- Multi-threaded crawling with configurable depth and delay
- Metadata, keywords, links, forms, images, and file extraction
- Technology detection
- SEO analysis
- SQLite database output
- JSON and CSV export
- Optional robots.txt respect
- Live logging for UI integration (Gradio)
